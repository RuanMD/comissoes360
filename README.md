# comissoes360

Projeto para gerenciamento de comissões 360.

## Como usar o script de upload

1. Dê permissão de execução (só precisa ser feito uma vez):
   ```bash
   chmod +x upload.sh
   ```

2. Execute o script passando a mensagem de commit:
   ```bash
   ./upload.sh "Mensagem do commit"
   ```
